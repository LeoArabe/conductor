# Conductor

[Portugues](#portugues) | [English](#english)

---

## Portugues

### Sobre
Motor de orquestracao para workflows de IA multi-agente. O Conductor gerencia o ciclo de vida de agentes de IA descartaveis, coordenando tarefas complexas de forma automatizada.

### Tecnologias
- TypeScript
- Node.js

### Como rodar
```bash
npm install
npm run build
npm start
```

### Status
Em desenvolvimento inicial (v0.1.0).
